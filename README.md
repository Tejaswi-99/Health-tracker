

# 🧠 MindPulse — Mental Health Tracker Wearable

A modern, responsive **landing page** for *MindPulse*, a conceptual mental health wearable that tracks stress, sleep, and emotional wellbeing.

Built using **HTML, CSS, and Vanilla JavaScript**, this project focuses on **clean UI, storytelling, and product presentation**.

---

## 🌐 Overview

MindPulse is designed as a **product showcase website** that communicates:

* What the product does
* How it works
* Key features
* User testimonials
* Call-to-action for pre-orders

This is a **frontend-only project** (no backend).

---

## ✨ Features

### 🎯 Hero Section

* Product introduction with strong messaging
* CTA buttons (Pre-order / Demo)
* Animated badge indicator
* Device UI mockup with wellness score

### 📊 Device Simulation UI

* Wellness score ring (SVG-based)
* Metrics:

  * Sleep
  * Heart rate
  * Stress
  * HRV
* Interactive mood selector

### 🧩 Features Section

* 6 feature cards including:

  * Stress detection
  * Sleep & mood correlation
  * Weekly AI reports
  * Therapist dashboard
  * Smart notifications
  * Privacy-first approach

### ⚙️ How It Works

* Step-by-step product flow:

  1. Wear device
  2. Sync app
  3. Log mood
  4. Get insights

### 📈 Stats Section

* Highlight metrics (user impact & adoption)
* Clean visual bar layout

### 💬 Testimonials

* Realistic user feedback cards
* Star ratings
* Avatar placeholders

### 🚀 Call-to-Action (CTA)

* Pre-order section with pricing
* Secondary research link

### 📱 Responsive Design

* Fully responsive layout
* Mobile-friendly UI
* Grid adapts to smaller screens
